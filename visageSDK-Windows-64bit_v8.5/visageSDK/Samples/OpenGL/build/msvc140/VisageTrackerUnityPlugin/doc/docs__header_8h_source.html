15  * The VisageTrackerUnityPlugin is a plugin that exposes VisageTracker's, VisageFaceAnalyser's and VisageFaceRecognition's functionalities
16  * for use in managed programming languages such as C# and Java. A prebuilt version of the plugin
17  * is provided in <em>bin/VisageTrackerUnityPlugin.dll</em>. It's use is demonstrated in
18  * VisageTrackerUnity and FacialAnimationUnity samples where a C# script is used to call functions
19  * exposed in this library. This project is designed to be integrated in other applications and
20  * cannot be run on its own.
21  *
22  * \section building Building the project
23  *
24  * Prerequisite to building the sample project is installed Visual Studio 2015.
25  *
26  * To build VisageTrackerUnityPlugin open the Visual Studio project located in Samples/OpenGL/build/msvc140/VisageTrackerUnityPlugin subfolder
27  * and choose Build Solution from the Build menu.
28  *
29  * \section implementation Implementation overview
30  *
31  * The project consists of a Visual Studio project with C++ source code and header files.
32  * Within the project VisageTrackerUnityPlugin is splitted in three source code files, one for each functionality it comprises: VisageTracker's, VisageFaceAnalyser's and VisageFaceRecognition's functionality.
33  * Exposed functionalities are described in the following files:
