7 /// <summary>
8 /// Provides high-level functions for using visage|SDK face recognition capabilities in Unity game engine.
9 /// It implements high-level wrapper functions from visage|SDK that enable usage of visage|SDK face recognition capabilities in Unity C# scripts.
10 ///
11 /// VisageFaceRecognition contains a face recognition algorithm capable of measuring similarity between human faces and recognizing a person's identity from frontal facial images (yaw angle approximately from -20 to 20 degrees).
12 ///
13 /// Similarity between two faces is calculated as difference between their face descriptors and returned as a value between 0 (no similarity) and 1 (maximum similarity).
14 ///
15 /// The face descriptor is a condensed representation of a face.It is an array of short values.
16 /// The dimension of the array is obtained using getDescriptorSize() function, from now on in the text referred to as DESCRIPTOR_SIZE.
17 /// For any two faces, the distance between descriptors is a measure of their similarity.
18 ///
19 /// For the purpose of face recognition, faces are stored in a gallery.The gallery is an array of face descriptors, with a corresponding array of names,
20 /// so that the gallery may contain n descriptors and a corresponding name for each descriptor.
21 /// To perform recognition, the face descriptor extracted from the input image is then compared with face descriptors from the gallery and the similarity between the input face descriptor
22 /// and all face descriptors from the gallery is calculated in order to find the face(s) from the gallery that are most similar to the input face.
23 /// The VisageFaceRecognition class includes a full set of functions for manipulating the gallery, including adding, removing, naming descriptors in the gallery as well as loading and saving the gallery to file.
24 ///
25 /// The functionality of VisageFaceRecognition is dependent on the VisageTracker and it is expected that prior to the call of VisageFaceRecognition's functions frame is processed with _track() function
26 /// and the data thus obtained is prepared and forwarded to VisageFaceRecognition.
27 ///
28 /// Before using VisageFaceRecognition, it must be initialized by calling the _initRecognition() function.
29 /// Afterward, in each frame, the _prepareDataForRecog() function is called first and then all other VisageFaceRecognition's functions can be called
30 /// otherwise they will return -1 as an indication that the necessary data is missing.
31 ///
32 /// VisageFaceRecognition's functionality can be used in a separate thread.
33 /// </summary>
34 ///

147 /** Compare a face to all faces in the current gallery and return n names of the most similar faces.
148 *
149 * Names array is passed as StringBuilder array to the function. The following code is an example of how the StringBuilder array is created, passed to the function and used after it is filled with strings:
150 * \code
151 *
152 * //maximum number of characters per name
153 * const int SB_CAPACITY = 50;
154 *
155 * //prepare stringbuilder
156 * System.Text.StringBuilder[] sbArray = new System.Text.StringBuilder[n];
157 * for (int i = 0; i < n; ++i)
158 * {
159 * sbArray[i] = new System.Text.StringBuilder(SB_CAPACITY);
160 * }
161 * //call of the _recognize function
162 * int ret = _recognize(descriptor, n, sbArray, SB_CAPACITY, similarities);
163 *
164 * List<string> names = new List<string>();
165 *
166 * //fill list of strings
167 * for (int i = 0; i < n; ++i)
168 * {
169 * names.Add(sbArray[i].ToString());
170 * }
171 *
172 * \endcode
173 *
174 * The example can be found in the VisageTrackerNative.Windows.cs (_recognizeWrapper() function) file within the VisageTrackerUnityDemo project.
175 *
176 * @param descriptor Face descriptor of the face to be recognized (compared to the gallery).
177 * @param n Number of names and similarities to be returned.
178 * @param names List type of <a href="https://msdn.microsoft.com/en-us/library/system.text.stringbuilder(v=vs.110).aspx">StringBuilder</a> that will be filled with the names of n faces from the gallery that are most similar to the input image ascending by similarity. Maximum number of characters per name is set to 256, for larger name only first 256 characters will be returned.
179 * @param capacity Number representing the desired capacity of a particular name in a string of names.
180 * @param similarities Array that will be filled with the similarity values for the n most similar faces, corresponding to the names array. The values are sorted, with the largest similarity value first.
181 * @return Number of compared faces.
182 */
183 int _recognize(short[] descriptor, int n, System.Text.StringBuilder[] names, int capacity, float[] similarities);
